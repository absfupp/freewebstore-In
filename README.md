# Freewebstore-In
This repository contains the front-end templates and UI projects developed during my internship at **Freewebstore**.

The goal of these projects is to strengthen front-end development skills by building responsive, modern, and reusable website templates using HTML, CSS, and JavaScript.

## 📁 Projects

This repository will continue to grow as more templates are completed.

| Project | Description | Status |
|---------|-------------|--------|
| Fitness Landing Page | Responsive one-page fitness website with interactive navigation and animations | ✅ Completed |
| More templates | Coming soon... | 🚧 |
